# R-Statistical-Inference

R code and solutions for assignments from **MATH682 - Statistical Inference** (Fall 2023).

## Contents

| File | Description |
|---|---|
| `TP1_Q10.R` | Custom CDF implementation and probability interval calculations for a piecewise distribution |
| `TP4.Rmd` | Analysis of BIXI bike-sharing trip durations using exponential and beta distribution fitting; MLE estimation |
| `A2RQuestionssolutions.Rmd` | Assignment 2 solutions covering Gaussian and log-normal distributions, Chebyshev inequality, and confidence intervals |
| `A3RSoutions.Rmd` | Assignment 3 solutions on uniform order statistics, chi-squared distribution simulation, and convergence in distribution |

## Topics Covered

- Probability distributions (Normal, Log-normal, Gamma, Beta, Chi-squared, Exponential)
- CDF construction and probability interval computation
- Maximum likelihood estimation (MLE)
- Chebyshev inequality
- Sampling distributions and simulation
- Confidence intervals

## Requirements

- R (≥ 4.0)
- R packages: `knitr`, `rmarkdown`
