# C Software Engineering

C programs and exercises from a software construction and maintenance course

## Requirements & Dependencies

### To Build
- **GCC** — C compiler (`gcc`)
- **GNU Make** — for running `make` commands

### To Test
- **[Bats](https://github.com/bats-core/bats-core)** — Bash Automated Testing System (used for shell-based test scripts)

### Platform
- Developed and tested on **Linux** (Ubuntu 16.04 LTS recommended)
- Should work on other Linux distributions and macOS
